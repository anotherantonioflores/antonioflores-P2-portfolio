We’ve analyzed which job types had the MOST positive outcomes, but now we want to compare which job type had the greatest PERCENT of positive outcomes.

posjob = mydata %>% 
  filter(y_termSubscribed==1) %>% 
  count(y_termSubscribed, job, sort=TRUE)

jobtable = mydata %>% count(job)

pctjob = posjob %>% 
  left_join(jobtable, by ="job")


colnames(pctjob)[3] <- "numofYES"
colnames(pctjob)[4] <- "numofBOTH"
p13 = ggplot(pctjob, aes(x=reorder(job, numofYES), y=numofYES, fill=numofYES))+
  geom_bar(stat="identity")+
  coord_flip()
  
p14 = ggplot(pctjob, aes(x=reorder(job, (numofYES/numofBOTH)), y=(numofYES/numofBOTH), fill=numofYES))+
  geom_bar(stat="identity")+
  coord_flip()

figure_file = here("results", "figures", "jobs-count-yes.png")
ggsave(filename = figure_file, plot=p13) 
Saving 7 x 5 in image

On the first figure, we can see that jobs with the most positive outcomes are management, technician, and blue-collar, but on the second chart we can see that the job-types with the highest percentages are ‘student’, ‘retired’, and ‘unemployed.’ There appears to be a noticable different between the total number of positive cases, and the job types which are most commonly recording positive cases.
